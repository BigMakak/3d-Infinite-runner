# 3D-Infinite-runner
A simple 3D Infinite Runner with Procedural Generation

## Controls
- Press Space to Jump
- The player wins Points for Jumping into new platforms, collecting coins and running!
- Don't fall!

## Settings
- Game made with Unity 2022.3.9f1
